## 介绍

一个用于播放电视直播流的多功能工具类App

## 已实现功能
- 添加远程m3u文件
- 播放m3u8文件
- 频道回看（需直播源支持）
- 实时获取当前频道预览图
- 获取频道节目单
- iCloud同步
- AppleTV 版本
