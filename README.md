# APTV
## TG交流群
> [t.me/AptvPlayer](https://t.me/AptvPlayer)
## 介绍

一个用于播放电视直播流的多功能工具类App

## 已实现功能
- 添加远程m3u文件
- 播放m3u8文件
- 实时获取当前频道预览图
- 获取频道节目单
- iCloud同步
- AppleTV 版本

## 接下来的计划
- Mac 桌面版本  
- CarPlay版本（或许有）
